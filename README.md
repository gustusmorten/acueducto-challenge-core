

## Getting Started

Make a copy of the variable example file and fill in the variables

```bash
cp .env.example .env


```
Run the development server:
```bash
docker-compose up -d
```
#### or
```bash
php artisan serve
```

run the migrations and seed the database
```bash
php artisan migrate --seed
```
